LectoSphere

LectoSphere est une application de gestion de bibliothèque développée avec TypeScript, React, Express et PostgreSQL.

* Fonctionnalités
Gestion des utilisateurs (membres de l’Alliance Française)
Suivi des emprunts et retours de livres
Catalogage des ouvrages
Intégration avec l’API Google Books pour enrichir automatiquement les données
Scan ISBN pour accélérer l’ajout de nouveaux livres
Gestion d’un catalogue de plus de 8000 exemplaires
* Stack technique
Frontend : React + TypeScript
Backend : Express + TypeScript
Base de données : PostgreSQL
* Objectif

Optimiser et moderniser la gestion des bibliothèques en automatisant le catalogage et en simplifiant le suivi des emprunts.
